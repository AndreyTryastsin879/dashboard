О приложении

Приложение собирает данные из разных источников и объединяет их в единный отчет с графиками.

В админке можно добавить новый проект, создать дашборд (например, SEO или SMM ) и привязать его к проекту, создать заметку и привязать ее к проекту.

Реализованные дашборды
SEO

Для каждого проекта собирается:
  • Количество визитов из Яндекса и Google (API Яндекс.Метрика) - ежедневно в 23:55

  • Количество проиндексированных страниц в Яндексе (API Яндекс.Вебмастер) - ежедневно в 23:00,

  • Количество проиндексированных страниц в Google (пасинг Google) - раз в неделю,

  • Доля отслеживаемых запросов в органической выдаче Яндекса и Google (API SERanking) - остановлено из-за санкций,

  • Количество страниц в Sitemap (парсинг Sitemap) - раз в неделю,

  • Количество визитов по каждому URL для расчета распределения трафика по страницам (API Яндекс.Метрика) - первого числа каждого месяца.

Отчет по каждой выгрузке можно посмотреть в разделе "Общая информация о проекте".
Админка

В админке можно создавать и редактировать:
  • Проект (Project) - обязательно нужно задать поле name и slug, иначе приложение упадет ;) Для сбора данных нужно задать соответсвующие остальные поля,

  • Дашборд (Service) - после заполнения всех полей привязать к нужному проекту,

  • Заметку (Note) - после заполнения всех полей привязать к нужному проекту.
